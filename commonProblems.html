Common Problems

Indic Keyboard settings doesnt opening while typing (MIUI)

This is due to a permission problem. This can be fixed.

  • Open settings
  • Go to Permissions
  • Navigate to other permissions
  • Select Indic Keyboard from the list
  • Give permission for Display pop-up windows while running in the background

Thats it.. You fixed it...
